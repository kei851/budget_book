# 家計簿アプリ 機能仕様書

**バージョン**: 1.0  
**作成日**: 2025年8月10日  
**更新日**: 2025年8月10日  

## 1. 概要

本仕様書は、楽天カードおよびエポスカード明細CSVファイルから支出データを可視化する家計簿アプリケーションの **機能詳細仕様** を定義する。要件定義書で定められた機能要件を、具体的な実装可能な仕様に落とし込んだものである。

**注意**: 本仕様書に記載されている金額、店名、件数等のデータは全てサンプル・例示用であり、実際のデータではありません。

## 2. 機能詳細仕様

### 2.1 CSVファイル読み込み機能

#### 2.1.1 ファイル選択機能
- **機能ID**: F001
- **概要**: ユーザーがCSVファイルを選択・アップロードする
- **入力**: 楽天カード/エポスカード明細CSVファイル
- **処理**:
  1. ドラッグ&ドロップまたはファイル選択ダイアログ
  2. ファイル形式検証（.csv拡張子チェック）
  3. ファイルサイズ検証（最大100MB）
  4. エンコーディング自動検出（UTF-8 with BOM）
- **出力**: ファイル読み込み成功/失敗メッセージ
- **エラー処理**: 
  - 非対応ファイル形式の場合はエラーメッセージ表示
  - ファイル破損の場合はエラーメッセージ表示

#### 2.1.2 CSVデータ解析機能
- **機能ID**: F002
- **概要**: CSVファイルのデータを解析して構造化データに変換
- **入力**: CSVファイルデータ
- **処理**:
  1. CSVヘッダー行の検証
  2. 必須カラムの存在確認（利用日、利用店名・商品名、利用金額）
  3. データ型変換（日付、数値）
  4. 不正データの検出・除外
- **出力**: 構造化された取引データ配列
- **データ形式**:
```javascript
{
  date: "2025-07-31",           // 利用日 (YYYY-MM-DD)
  store: "JCB サイゼリヤ",       // 店名・商品名
  amount: 770,                  // 利用金額
  paymentMethod: "1回払い",      // 支払方法
  user: "本人",                 // 利用者
  category: null                // カテゴリ（後で自動分類）
}
```

### 2.2 カテゴリ自動分類機能

#### 2.2.1 パターンマッチング分類
- **機能ID**: F003
- **概要**: 店名・商品名からカテゴリを自動判定
- **入力**: 取引データ（店名・商品名）
- **処理**:
  1. 事前定義されたキーワード辞書との照合
  2. 正規表現パターンマッチング
  3. カテゴリ判定ロジックの適用
- **出力**: カテゴリ分類済み取引データ

#### 2.2.2 分類ルール定義
- **カテゴリ一覧**:
  - 食費: スーパー、コンビニ、レストラン、カフェ等
  - 交通費: JR、私鉄、バス、タクシー、ガソリンスタンド等  
  - 住宅費: 家賃、管理費、修繕費等
  - 光熱・通信費: 電力会社、ガス会社、携帯キャリア、ISP等
  - 娯楽: 映画館、ゲーム、スポーツ施設等
  - 投資・金融: 証券会社、保険会社、銀行手数料等
  - ショッピング・サービス: 衣料品店、家電量販店、美容院、病院等
  - その他: 上記に該当しないもの

#### 2.2.3 手動修正機能
- **機能ID**: F004
- **概要**: 自動分類結果の手動修正
- **入力**: 分類済み取引データ、ユーザー修正操作
- **処理**:
  1. 個別取引のカテゴリ変更（ドロップダウン選択）
  2. 複数取引の一括カテゴリ変更
  3. 修正履歴の記録
  4. ユーザー学習機能（同じ店舗の分類を記憶）
- **出力**: 修正済み取引データ

### 2.3 データ処理・集計機能

#### 2.3.1 月次データ集計
- **機能ID**: F005
- **概要**: 取引データを月毎・カテゴリ毎に集計
- **入力**: カテゴリ分類済み取引データ
- **処理**:
  1. 利用日による月毎グルーピング
  2. カテゴリ毎の金額合計計算
  3. 統計データ生成（合計、平均、最大、最小、件数）
- **出力**: 月次集計データ
```javascript
{
  "2025-07": {
    "食費": 32458,
    "交通費": 10000,
    "住宅費": 0,
    "光熱・通信費": 8899,
    "娯楽": 14800,
    "投資・金融": 48438,
    "ショッピング・サービス": 66183,
    "その他": 6768,
    "合計": 187546,
    "件数": 106
  }
}
```

### 2.4 グラフ表示機能

#### 2.4.1 積み上げ棒グラフ表示
- **機能ID**: F006
- **概要**: 月毎・カテゴリ別支出を積み上げ棒グラフで表示
- **入力**: 月次集計データ
- **処理**:
  1. Chart.jsによるグラフ描画
  2. カテゴリ別色分け表示
  3. インタラクティブ機能（ホバー、クリック）
  4. 凡例表示
- **出力**: 積み上げ棒グラフUI

#### 2.4.2 期間選択機能
- **機能ID**: F007
- **概要**: グラフ表示期間の切り替え
- **入力**: 期間選択（3ヶ月/6ヶ月/12ヶ月/全期間）
- **処理**:
  1. 選択期間に基づくデータフィルタリング
  2. グラフの再描画
- **出力**: 期間限定グラフ表示

#### 2.4.3 カテゴリ表示制御（オプション）
- **機能ID**: F008
- **概要**: 特定カテゴリの表示・非表示切り替え
- **入力**: カテゴリ選択操作
- **処理**:
  1. 凡例クリックによるカテゴリ非表示
  2. 設定保存・復元
- **出力**: カスタマイズされたグラフ表示

### 2.5 詳細表示・編集機能

#### 2.5.1 取引明細一覧表示
- **機能ID**: F009
- **概要**: 取引明細をテーブル形式で表示
- **入力**: 分類済み取引データ
- **処理**:
  1. テーブル形式での明細表示
  2. ページネーション（100件/ページ）
  3. ソート機能（日付、金額、カテゴリ順）
- **出力**: 取引明細テーブルUI

#### 2.5.2 フィルタ・検索機能
- **機能ID**: F010
- **概要**: 取引明細の絞り込み・検索
- **入力**: フィルタ条件（カテゴリ、期間、金額、店名）
- **処理**:
  1. 条件に基づくデータフィルタリング
  2. 検索結果の表示
  3. フィルタ履歴管理
- **出力**: フィルタ済み明細一覧

#### 2.5.3 統計情報表示
- **機能ID**: F011
- **概要**: カテゴリ別円グラフと統計サマリー表示
- **入力**: 月次集計データ
- **処理**:
  1. カテゴリ別円グラフ描画
  2. 統計サマリー計算（1日平均、取引回数、最大・最小支出）
- **出力**: 分析結果UI

### 2.6 表示カスタマイズ機能（オプション）

#### 2.6.1 カラム表示制御
- **機能ID**: F012
- **概要**: 明細一覧の表示項目カスタマイズ
- **入力**: カラム選択操作
- **処理**:
  1. 表示項目の選択（利用日、店名、金額、カテゴリ等）
  2. カラム並び順の変更
  3. 設定の保存・復元
- **出力**: カスタマイズされた明細一覧

## 3. データフロー

```
CSVファイル → 読み込み・解析 → カテゴリ自動分類 → 手動修正 → 月次集計 → グラフ表示
                                      ↓
                                   詳細表示・編集
```

## 4. エラーハンドリング

### 4.1 ファイル読み込みエラー
- ファイル形式不正
- ファイルサイズ超過
- ファイル破損

### 4.2 データ処理エラー
- 必須カラム不存在
- データ型不正
- 日付形式不正

### 4.3 システムエラー
- メモリ不足
- 処理タイムアウト
- ブラウザ互換性エラー
