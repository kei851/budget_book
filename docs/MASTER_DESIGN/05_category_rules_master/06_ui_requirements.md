## 1. UI 要件概要

### 1.1 ページ構成

**CategoryRulesPage** は、分類ルールの管理を行うページです。

**主要機能**:
- 📋 ルール一覧表示
- ➕ ルール追加
- ✏️ ルール編集
- 🗑️ ルール削除
- 📊 マッチング件数表示

---

## 2. ルール一覧表示

### 2.1 表示項目

| カラム | 説明 | データソース |
|--------|------|------------|
| キーワード | マッチング対象キーワード | `keyword` |
| カテゴリ | 割り当てカテゴリ | `category_name`, `category_color` |
| 優先度 | 優先度 | `priority` |
| マッチング件数 | このキーワードにマッチする取引件数 | 計算値 |
| 操作 | 編集・削除ボタン | - |

### 2.2 表示順序

- **デフォルト**: 優先度降順（`priority DESC`）
- **オプション**: カテゴリ別フィルタ、キーワード検索（将来的に検討）

---

## 3. ルール追加・編集・削除フォーム

### 3.1 ルール追加フォーム

**フィールド**:
- **キーワード**: テキスト入力（必須）
- **カテゴリ**: ドロップダウン選択（必須）
- **優先度**: 数値入力（必須、デフォルト: 50）

### 3.2 ルール編集フォーム

**機能**: インライン編集（テーブル内で直接編集）

**フィールド**: 追加フォームと同様

### 3.3 ルール削除

**機能**: 削除ボタンクリックで確認ダイアログ表示後、削除
