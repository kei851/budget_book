# Figma画面設計ガイド

## 1. Figmaプロジェクト準備

### 1.1 新規プロジェクト作成
1. Figmaにログイン
2. 新規ファイル作成: "家計簿アプリ UI設計"
3. フレーム設定: Desktop (1440 x 1024) とMobile (375 x 812)

### 1.2 カラーパレット設定
機能仕様書のカラー定義をFigmaのカラースタイルに登録：

**カテゴリ色**
- 投資・金融: #FF6384
- 食費: #4BC0C0  
- 娯楽: #36A2EB
- 交通費: #FFCE56
- 光熱・通信費: #FF9F40
- ショッピング・サービス: #9966FF
- 住宅費: #FF6B6B
- その他: #C9CBCF

**UI色**
- Primary: #4CAF50 (メインカラー)
- Success: #28a745
- Danger: #dc3545
- Warning: #ffc107
- Info: #17a2b8
- Background: #f8f9fa
- Text: #495057

### 1.3 テキストスタイル設定
- Heading 1: 32px Bold (ヘッダータイトル)
- Heading 2: 24px Bold (セクションタイトル)
- Heading 3: 18px Bold (カードタイトル)
- Body: 16px Regular (通常テキスト)
- Caption: 14px Regular (補足テキスト)
- Small: 12px Regular (ラベル)

## 2. コンポーネント作成

### 2.1 基本コンポーネント
**ボタン**
- Primary Button: #4CAF50背景、白文字、角丸5px
- Secondary Button: #e9ecef背景、黒文字、角丸20px（期間選択用）
- Upload Button: 点線枠、ホバー効果付き

**カード**
- Stat Card: 200x100px、角丸10px、シャドウ、グラデーション背景
- Info Card: 白背景、角丸10px、軽いシャドウ

**フォーム要素**
- Dropdown: 白背景、1px solid #dee2e6、角丸5px
- Search Input: プレースホルダー付き、アイコン

### 2.2 グラフプレースホルダー
Chart.jsの実装予定部分は以下でプレースホルダー作成：
- 積み上げ棒グラフエリア: 800x400pxの長方形
- 円グラフエリア: 200x200pxの円
- 簡単な棒や円の図形でサンプル表示

## 4. Figma作成手順

### Step 1: フレーム作成
1. メインページ用フレーム作成 (Desktop)
2. 詳細ページ用フレーム作成 (Desktop)
3. モバイル版フレーム作成 (Mobile) 

### Step 2: コンポーネント作成
1. ヘッダーコンポーネント
2. アップロードエリアコンポーネント
3. 統計カードコンポーネント（4つのバリエーション）
4. ボタンコンポーネント（プライマリ、セカンダリ）
5. テーブル行コンポーネント

### Step 3: グラフプレースホルダー
1. 積み上げ棒グラフ: 長方形 + サンプル棒（色付き）
2. 円グラフ: 円 + セクター分割（色付き）
3. 凡例: 色円 + テキストのセット

### Step 4: レスポンシブ対応
1. タブレット版 (768px)
2. モバイル版 (375px)
3. ブレークポイント別のレイアウト調整
